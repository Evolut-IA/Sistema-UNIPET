Informações Básicas:
- Nome do pet*
- Espécie*
- Raça
- Data de nascimento ou Idade
- Sexo*
- Castração/esterilização (sim/não)
- Cor/pelagem
- Peso atual

Cuidados Preventivos:
- Datas de vacinas (raiva, V10, etc.)
- Último check-up
- Tratamentos antipulgas/parasitas

#2 Guias de atendimento
Para gerar a guia, deve ter um campo para digitar para encontrar o cliente. Utilize o CPF, nome, e-mail ou telefone informado no cadastro do cliente. Após localizar o cliente, selecione um dos pets vinculados ao seu cadastro para prosseguir.
Tambem deve ter o campo onde será possivel selecionar qual o procedimento da guia aberta de acordo com as possibilidades do plano que o pet está vinculado.
Deve ter um campo textarea para anotar sobre o procedimento e um campo textarea para anotações gerais. Os campos de textarea não devem ser obrigatórios, já os outros sim.

Tipos de guias que podem ser abertas:
- Guia de Consulta
- Guia de Exames
- Guia de Internação
- Guia de Reembolso

#3  Gestão de Procedimentos
Configuração de Planos e Valores:
O sistema permite definir quais procedimentos estão inclusos em cada plano, com valores específicos por procedimento de acordo com o plano selecionado.

Exemplo de Precificação:
- Plano Básico: Consulta Clínica Geral = R$ 50,00
- Plano Confort: Consulta Clínica Geral = R$ 20,00

Seções de Procedimentos Disponíveis:
Para cada plano, é possível configurar os benefícios nas seguintes seções. Caso alguma seção não seja utilizada em determinado plano, ela pode permanecer vazia e não será exibida.
- Consultas
- Procedimentos e Anestesia
- Procedimentos Ambulatoriais
- Exames Laboratoriais
- Exames Laboratoriais Complexos
- Cirurgia Eletiva
- Cirurgias
- Exames de Imagem
- Vacinas
- Benefícios Especiais
- Especialistas
